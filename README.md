# AstraTask Engine

Bot de Telegram para centralizar tareas y recordatorios usando Make + Google Sheets + Gemini (IA).

## Qué hace
- Recibe tareas por Telegram en lenguaje natural.
- Convierte el mensaje a JSON estructurado.
- Guarda y actualiza tareas en Google Sheets.
- Comandos: /registrar, /resumen, /hecho (según flujo).

## Stack
- Make.com
- Telegram Bot
- Google Sheets
- Gemini + parseo JSON
